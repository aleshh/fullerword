FullerWord

Currently under development, FullerWord is a personal dictionary app intended for writers.

Ever looked up a new word and wanted to use it in your writing? Save the default definition or your own definition and tag the word. Once you've got a collection the list can be filtered by tag to find just what you're looking for.

Currently designed for phone-sized screens. Uses browser local storage to store data.

Built by Alesh Houdek with Angular. Uses the Oxford Dictionary API and Feather icons.

Visit the GitHub repo for more information.
